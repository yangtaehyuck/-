# 상수와 변수 # 상수 → 중복 선언, 정의 불가능 const 식별자 = '자료' 형태로 작성 // 식별자를 상수로 선언한다, 정의한다라고 읽는다 const pi → 이렇게 못 씀 처음부터 값을 지정해줘야함 const pi = 3.141592 // 이렇게 처음으로 pi를 선언하고 값을 할당하는것을 '초기화'라고 한다 → const pi = 3.1412 재할당 불가능, 변경 불가능 ------------------------------------------------------------------------------------- # 변수 → 변경 가능 let 식별자 = '자료' // 식별자를 변수로 선언한다, 정의한다라고 읽는다. let 식별자 // 이렇게 초기화를 안해주면 값이 undefined(정의되지 않은)로 나옴 식별자 = '자료' // 식별자에 자료를 할당한다라고 읽는다 let pi → 이렇게 쓸 수 있음 처음부터 값을 안지정해줘도됨 pi = 3.14 → 처럼 나중에 할당가능 let pi = 3.14 → pi = 3.141592 재할당 가능 하지만 → 이렇게 2개를 같이 선언 할 수 없음 → let pi = 3.14 → let pi = 3.145 차라리 값을 재할당해주자 → let pi = 3.14 → pi = 3.145 # 상수와 변수의 사용 범위 상수는 기본적으로 사용 변수는 많이 변경될만한것을 변수로 사용 # lvalue rvalue let a = 10 a → left value → lvalue → 값을 넣은 놈, 할당받는 놈 10 → right value → rvalue → 값, 꺼내는 놈, 할당하는 놈 let b = a b → lvalue a → rvalue ex) let a a = 10 은 되는데 10 = 20 은 왜 안되냐? 위에 10은 rvalue인데 rvalue는 값, 할당하는 놈에다가 20을 할당하면 당연히 안되지 왼쪽에는 반드시 할당받는놈 넣는 놈이 와야한다 # 변수에 적용할 수 있는 연산자 # 복합 대입 연산자 let a = 100 a = a + 100 → a += 100 → a = 200 a -= 100 → a = a - 100 → a = 100 이 외에도 *= /= %= 등 다양한 복합 대입 연산자가 있다 복합 대입 연산자는 변수(let)에만 적용 할 수 있다. const a = 100으로 하면 안되요 let a = 100으로 해야함 # 증감 연산자 → 이것도 변수에만 가능함 // a++ a-- → a+=1, a-=1 과 같음 // ++a --a let a = 0 a++ → 현재 문장을 실행한 후에 a += 1 let a = 0 console.log(a++) → 0 console.log(a) → 1 ++a → 현재 문장을 실행하기 전에 a += 1 let b = 0 console.log(++b) → 1 console.log(b) → 1 a++ → a => 1 a-- → a => 0 # undefined 자료형 1. 상수와 변수로 선언하지 않은 식별자 let a를 선언하지 않고 아래와 같이 입력하면 typeof(a) → "undefined" 2. 값이 없는 변수 let b // b에 값을 할당하지 않으면 typeof(b) → "undefined"
